# Datasets

## Manga109-s

Manga109-s is used for box detection training/evaluation in this project.

- Official dataset homepage: [Manga109](http://www.manga109.org/en/)
- Dataset images are not redistributed in this repository.
- Any training/testing with Manga109-s requires downloading the dataset from
  the official source and complying with its license/usage terms.
